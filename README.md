## ğŸ“¥ Installation Guide
You don't need to use terminal or command line.

1.  **Download** the latest `VonCMS_Deploy.zip`.
2.  **Upload** the file to your hosting (cPanel/DirectAdmin file manager) or Localhost (`htdocs/`).
3.  **Extract** the zip file.
4.  **Open** your website URL (e.g., `www.yourdomain.com`).
5.  The **Install Wizard** will start automatically:
    *   Enter Database Details (Host, Name, User, Pass).
    *   Create Admin Account.
6.  **Done!** Your site is live.
