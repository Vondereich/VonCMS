> **The Hybrid CMS for the Modern Web.**
> Performance of a Static Site, Flexibility of a Headless CMS, Ease of WordPress.

VonCMS "Nara" is a hybrid content management system built for high-traffic portals. It combines a blazing-fast **React 19** frontend with a hardened **PHP 8.2** backend to deliver enterprise-grade speed (LCP 0.40s) on standard shared hosting.

> [!WARNING]
> **🚨 CRITICAL UPDATE NOTICE (v1.11.12)**
> Users on **v1.11.11 or older** MUST perform a **MANUAL UPDATE**. Delete the `public/assets` folder before uploading new files. Do NOT use the OTA Updater for this version.

---

## 🏛️ The VonCMS Philosophy

1.  **Security First**: 6-layer defense (CSRF, session binding, rate limiting, etc.) baked into a single `security.php` middleware.
2.  **Easiness Second**: 2-minute setup with zero-config essentials.
3.  **Flexibility Third**: Runs anywhere (cPanel, Shared Hosting, VPS) with a < 15MB memory footprint.

---

## 🛠️ The Technology Stack

- **Frontend**: React 19 + Vite 7 + TailwindCSS (Single Page Application experience)
- **Backend**: PHP 8.2+ (Universal compatibility, high performance)
- **Database**: MySQL/MariaDB (Optimized for millions of rows)
- **Editor**: TipTap WYSIWYG (Professional editing experience)

## ⚙️ Requirements & Quick Start


- **PHP**: 8.1+ (Optimized for 8.2) | **DB**: MySQL 5.7+
- **Server**: Apache / Litespeed (Supports .htaccess)

1. **Download**: Grab `VonCMS_v1.11.12_Deploy.zip`.
2. **Upload**: Extract to your root or subfolder.
3. **Install**: Navigate to `yoursite.com/install.php`.
4. **Publish**: Access `/admin` and start creating!
