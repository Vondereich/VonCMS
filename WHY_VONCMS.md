# VonCMS Features

VonCMS is designed as a battery-included CMS for teams, creators, publishers, and founders who want a modern publishing stack without moving to a Node-only hosting model.

It is not a bare starter kit. Out of the box, VonCMS ships with a working admin panel, publishing workflow, SEO stack, media tools, update flow, and multiple themes so a new install already feels like a real product instead of an empty framework.

## Why This Matters

Many CMS projects ask users to assemble the basics themselves. VonCMS takes the opposite approach. The core package already includes the tools most content-driven sites expect on day one:

- content management
- SEO foundations
- media handling
- comments and discussion
- newsletter flow
- theme switching
- updater and recovery tools
- built-in plugins for common growth features

That makes VonCMS suitable for people who want to launch faster, validate ideas earlier, and spend less time wiring essential features together from scratch.

## What Ships In The Core

### Publishing And Admin

VonCMS includes the core administration and publishing workflow needed to run a modern content site:

- installer wizard
- admin dashboard
- settings manager
- post manager
- page manager
- user manager
- discussion and comment moderation
- media manager and gallery
- database manager
- security dashboard
- contact forms manager
- newsletter manager
- extensions manager
- login and session-based authentication
- author and profile pages

### SEO And Discoverability

SEO is not treated as an afterthought. The core package already includes the foundations needed for search and social sharing:

- dynamic `sitemap.xml`
- dynamic `robots.txt`
- dynamic `llms.txt`
- canonical URL handling
- SEO-aware permalink structures
- redirect manager
- redirect engine
- Open Graph and social meta output
- JSON-LD schema output
- IndexNow support

### Media And Site Operations

VonCMS also includes practical maintenance and media tooling that many smaller CMS projects leave to external plugins:

- image upload pipeline
- WebP support
- media optimization settings
- thumbnail regeneration
- orphan media cleanup
- CDN URL support for new uploads
- OTA updater
- Integrity Check
- `Repair .htaccess`
- database repair utility
- WordPress XML migrator

### Engagement And Monetization

The default package already covers the common engagement and monetization layer for publishing sites:

- native comments system
- newsletter subscribe widget
- newsletter subscriber manager
- contact form submission flow
- ad slots for header, in-feed, and popup placements

### AI And Smart Features

VonCMS includes AI-oriented tooling in the shipped core as well:

- AI Write endpoint
- AI Check endpoint
- AI Summary plugin
- AI-ready site language settings

## Built-In Plugins

The core package currently includes these built-in plugins:

- Promo Bar
- Floating Gift Widget
- VonSEO
- VonAnalytics
- AI Summary
- Related Posts

These plugins extend the default install without forcing users to hunt for external add-ons before the site feels complete.

## Included Themes

VonCMS ships with multiple themes so users can start with a visual direction immediately:

- Default
- TechPress
- Prism
- Digest
- Portfolio
- Corporate Pro

This makes the platform useful across several common use cases, including blogs, news portals, magazine-style sites, creative portfolios, and business-facing websites.

## Who VonCMS Is For

VonCMS is a strong fit for:

- indie founders who want to launch a serious content product quickly
- agencies that need a self-hosted CMS with a more modern admin experience
- publishers who want built-in SEO, themes, and growth tools from the start
- teams that prefer PHP hosting practicality without giving up a modern React-based frontend

## Summary

VonCMS is best understood as a full CMS product, not a thin shell. The default package already includes enough functionality to be useful in production, while still leaving room for marketplace, premium, hosted, or VIP layers on top.
